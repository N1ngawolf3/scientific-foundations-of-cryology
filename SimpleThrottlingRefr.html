Необходимо определить для:
Рефрижераторных циклов - удельную холодопроизводительность, холодильный
коэффициент и степень термодинамического совершенства цикла;

Для ожижительных
циклов - коэффициент ожижения, затраты энергии для получения 1кг жидкого продукта и
степень термодинамического совершенства.

Цикл Простого Дросселирования

Рефрижераторный режим

Дано: {{fluid}}  p2 = [p1,p2]*105 Па   T1 = {{T1}} К

Решение

T3 = Tx = {{Tx}} K  h1=[h1,h1] КДж/кг  s1 = [s1,s1] КДж/кг*K

T4 = T3 = {{Tx}} K  h4=[h4,h4] КДж/кг  s4 = [s4,s4] КДж/кг*K

T5 = T1 - ΔTн = {{T5}} h5=[h5,h5] КДж/кг  s5 = [s5,s5] КДж/кг*K

T6 = T1 = {{T1}} h6=[h6,h6] КДж/кг  s6 = [s6,s6] КДж/кг*K

Полезная холодопроизводительность: qx = h5 - h1 - qoc = {{qx}} КДж/кг

Затр. работа на сжатие в компр.: lсж = T1*(s6-s1)-(h6-h1)/ηиз = {{l_compr}} КДж/кг

Холодильный коэффициент: ε = qx/lсж = [ε ,ε ]

Холодильный коэффициент цикла Карно: εк = T4/T1-T4 = {{refr_coef_Carno}}

Степень термодинамического совершенствования: ηT = ε/εк = [ηT ,ηT ]
